## Obsidian

| Action                      | Shortcut              |
| --------------------------- | --------------------- |

| Export to PDF               | `⌘` + `⇧` + `E`       |
| Pin tab                     | `⌘` + `.`             |
| Close all other panes       | `⌘` + `⇧` + `W`       |

| Find unlinked files         | `⌘` + `U`             |

| Focus on pane below         | `⌘` + `↓`             |
| Focus on pane to the left   | `⌘` + `⎇` + `←`       |
| Focus on pane to the right  | `⌘` + `⎇` + `→`       |
| Use dark mode               | `⎇` + `D`             |
| Use light mode              | `⎇` + `L`             |
| Toggle left sidebar         | `⎇` + `←`             |
| Toggle right sidebar        | `⎇` + `→`             |
| Navigate back               | `⌘` + `←`             |
| Navigate forward            | `⌘` + `→`             |
| Move file to another folder | `⌘` + `M`             |
